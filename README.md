<div class="collapsible-block-content">
<div style="text-align: left;">
<p>欢迎来到SCP基金会中文wiki。在发布你的第一篇原创作品或者翻译作品之前，请先确保你已经做过以下事情：</p>
<ol>
<li>确保读遍了<a href="/site-rules">站规</a>和<a href="/guide-for-newbies">新人指南</a>。</li>
<li><a href="/system:join">加入</a>成为SCP中文维基的会员。</li>
<li>在<a href="http://scpsandboxcn.wikidot.com/">SCP中文沙盒维基</a>里为你的草稿腾一片地方。</li>
<li>利用<a href="/forum:start">本站的讨论区</a>或者各种聊天群在发布前为你的草稿找点意见和批评。</li>
</ol>
<div class="contribute"><script type="text/javascript" src="https://d3g0gp89917ko0.cloudfront.net/v--3e3a6f7dbcc9/common--javascript/yahooui/tabview-min.js">

<div class="yui-content">
<div id="wiki-tab-0-0">
<p><strong>SCP文档</strong>是基金会的核心所在，是一系列描述了各种由基金会收容的异常项目、个体、以​​及现象的技术性文章。</p>
<p>请阅读<a href="/how-to-write-an-scp">如何撰写一篇SCP文档</a>来了解如何撰写并发布一份SCP文档。SCP社区对于腔调和技术性文本的准确性有着很严格的标准；强烈建议你去读任何和你的点子有关的<a href="/guide-hub">指导和文章</a>，并在发布前使用<a href="/forum:start">本站的讨论区</a>或者各种聊天群汲取意见和批评。</p>
<ul>
<li><strong><a href="/how-to-write-an-scp">如何撰写一篇SCP文档</a></strong></li>
<li><strong><a href="/scp-series-cn">SCP-CN系列 I</a></strong></li>
<li><strong><a href="/scp-series-cn-2">SCP-CN系列 II</a></strong></li>
</ul>
</div>
<div id="wiki-tab-0-1" style="display:none">
<p>如果说SCP文档是基金会的核心所在，那么<strong>故事</strong>就是它的灵魂。故事可以是一份正式的报告，一篇叙事性的短故事，或者一首诗等等；它们能够展示基金会的一部份面貌，在基金会里的人们，或在基金会世界观下的其他话题。</p>
<p>你可以通过下面的“新原创故事”栏创建一个原创故事。</p>
<div style="font-size: 80%; font-style: italic; color: #c69; line-height: 130%;">
<p><strong>注意：</strong>用此栏创建的页面将会自动添加“<em>原创</em>”及“<em>故事</em>”标签。请用底下的通用页面栏去创建非“<em>故事</em>”页面。</p>
</div>
<ul>
<li><strong><a href="/foundation-tales-cn">基金会中国分部故事</a></strong></li>
</ul>

<div id="wiki-tab-0-2" style="display:none">
<p><strong>另类文档</strong>是有着很多和SCP文档一样的框架特征，但在主题及意图上有差异的副系列。虽然在写作难度上另类文档比常规的SCP文档难上许多倍，但它们都能在基金会式的叙事风格上擦出各自的火花。</p>
<ul>
<li><strong><a href="/joke-scps-cn">搞笑SCP-CN</a></strong></li>
<li><strong><a href="/scp-ex-cn">已解明SCP-CN</a></strong></li>

<li><strong><a href="/log-of-extranormal-events-cn">中国分部超常事件记录</a></strong></li>
</ul>
</div>
<div id="wiki-tab-0-3" style="display:none">
<p><strong>设定</strong>是合作的故事系列，可以探索基金会世界观中一个特定的方面，其他相关组织（GoI），平行宇宙，或者各种不同的演绎。一个设定下的原创作品，其创建方式与原创故事相同。</p>
<p>我们也允许在中文站上为英语站乃至其他国际分部的设定创作原创中文作品。</p>
<ul>
<li><strong><a href="/canon-hub">设定中心</a></strong></li>
<li><strong><a href="/canon-hub-cn">中国分部设定中心</a></strong></li>
</ul>
</div>
<div id="wiki-tab-0-4" style="display:none">
<p>本站运行的目的之一，就是将英语站的作品，以至其他分部的外语作品<strong>翻译</strong>成汉语，以方便使用汉语的读者进行阅读。</p>
<p>除了翻译外语作品外，另一个重要的工作就是从中文论坛及百度贴吧等地将以前已翻译的文档搬运到维基上。</p>
<p>在发布或搬运前，请先阅读<a href="/translation-movement-guide">翻译发布及搬运指南</a>来了解如何发布或搬运一篇翻译，以减轻工作人员在整理翻译时的负担。</p>
<ul>
<li><strong><a href="/translation-movement-guide">翻译发布及搬运指南</a></strong></li>

<li><strong><a href="/scp-international">SCP国际翻译中心</a></strong></li>
</ul>

<p>你可以从<a href="http://wanderers-library.wikidot.com/">英语站</a>翻译既有的作品，也可以创作中文的原创篇章。在这里写作的题材和格式非常自由，任何形式的作品都可以受到欢迎。</p>
